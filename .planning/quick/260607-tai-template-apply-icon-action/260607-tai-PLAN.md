# Template Apply Icon Action

## Goal

Move the visible apply-template action beside the template overflow dropdown and make the apply action icon-only.

## Scope

- Keep apply visible by default.
- Keep the overflow dropdown as the top-right card control.
- Preserve existing apply, preview, edit, and delete behavior.

## Verification

- Run `yarn build`.
