<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Shared manifest to local Redux state | User-selected shared updates mutate local shared cookbook data. |
| Shared data fetch to modal action | Final sync must not run until required data has loaded successfully. |
| Remote image URL to list row | Remote image fetch/decode can consume browser resources while rows remain interactive. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-T | Tampering | Selective sync versions | mitigate | Merge synced version updates with existing stored values and test one-category sync. |
| T-03-02-D | Denial of Service | Sync modal fetch/impact work | mitigate | Render shell first, schedule full fetch/warnings, and disable only final sync until data is safe. |
| T-03-02-D2 | Denial of Service | Dish image request/decode | mitigate | Use fallback-first list mode and controlled image tests for slow/blocked modes. |
</threat_model>
